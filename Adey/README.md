# Adey CPCA - Official Website

![Adey CPCA Logo](assets/images/LOGO%20ADEY.jpg)

**Adey Cerebral Palsy Charitable Association (CPCA)** is a non-profit organization licensed by the Authority for Civil Society Organizations (ACSO) in Ethiopia. Established in August 2024, the association is dedicated to improving the lives of children living with Cerebral Palsy and providing critical support systems for their families and caregivers.

This repository contains the source code for the official website: [adeycpca.org](https://adeycpca.org)

## 🌟 Mission
Our digital platform bridges the gap between those in need and those willing to help. It serves as a comprehensive resource for the Ethiopian community, offering information on:
- **Therapeutic Interventions**
- **Specialized Education**
- **Assistive Device Provision**

## 🛠️ Technical Stack
The website is built with a focus on premium aesthetics, performance, and reliability:
- **HTML5 & Vanilla CSS**: For structure and bespoke styling.
- **Tailwind CSS**: A utility-first CSS framework for a responsive, modern design system.
- **JavaScript (ES6+)**: Powers dynamic UI components and secure payment logic.
- **Chapa Payment Gateway**: Integrated secure donation flow supporting Telebirr, CBE Birr, eBirr, mPesa, and USSD.
- **Lucide Icons**: Clean, expressive iconography for visual navigation.

## 🚀 Key Features
- **Modern Responsive Design**: Optimized for mobile, tablet, and desktop users.
- **Secure Donation Module**: Robust hosted checkout integration with Chapa API.
- **Transparency & Trust**: Detailed sections for **Safeguarding Policies**, **Who We Are**, and **Our Impact**.
- **Bank Transfer Integration**: Easy-to-use copy features for direct bank contributions.

## 📦 Deployment
The site is hosted on **Netlify** and is automatically deployed via this GitHub repository.
- **Production Branch**: `main`
- **Domain**: `adeycpca.org`
